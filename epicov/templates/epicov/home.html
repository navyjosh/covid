ECES-450 Group Project

The scope of this project is to use covid-19 data from GISAID along with metadata from Nextstrain in order to build a visualization that describes the geographical spread of covid-19 virus around the world with respect to time. This project will implement various tools in python and/or javascript to build the visualization.

GISAID

GISAID is a public–private partnership between the German government and the nonprofit organization Friends of GISAID that provides public access to the most complete collection of genetic sequence data of influenza viruses and related clinical and epidemiological data through its database (named EpiFlu).[1] The database is hosted by the German government.
