ECES-450 Group Project

The scope of this project is to use covid-19 data from GISAID along with metadata from Nextstrain in order to build a visualization that describes the geographical spread of covid-19 virus around the world with respect to time. This project will implement various tools in python and/or javascript to build the visualization.
